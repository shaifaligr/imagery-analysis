File named 'data' can be used to automatically download both Sentinel-1 and Sentinel-2 using the Copernicus account credentials. 
Sentinel-1 preprocessing can be done in SNAP. 
File named 'S1' can be used to perform change detection on the preprocessed datasets. 
File 'S2' can be used to perform change detection on the Sentinel-2 products. requirements.yml gives the required libraries. 
NDVI is generated for both the files of Sentinel-2. 
Change mask is also generated for S1 and S2 images.
